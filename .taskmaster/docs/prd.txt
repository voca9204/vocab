# Vocabulary - SAT 영어 학습 앱

## 프로젝트 개요
고등학교 10-11학년 학생들을 위한 SAT 영어시험 대비 학습 앱입니다. 단어, 문법, 독해를 종합적으로 학습할 수 있는 웹 애플리케이션을 개발합니다.

## 목표 사용자
- 고등학교 10-11학년 학생 (15-17세)
- SAT 영어 시험 준비생
- 영어 어휘력 향상을 원하는 학습자

## 핵심 기능

### 1. 단어 학습 시스템
- **단어 퀴즈**: 다양한 형태의 단어 시험 (객관식, 단답형, 매칭)
- **실전 예문**: 유명한 문장, 뉴스 기사에서 가져온 실제 예문
- **맥락 학습**: 단어가 사용되는 다양한 상황과 문맥 제공
- **난이도별 분류**: SAT 빈출 단어를 난이도별로 분류

### 2. 발음 기능
- **음성 재생**: 모든 단어의 정확한 발음 제공
- **발음 연습**: 사용자가 따라 말할 수 있는 인터랙티브 기능
- **발음 평가**: 사용자 발음의 정확도 측정 (선택적)

### 3. 문법 학습
- **문법 문제**: SAT 형태의 문법 문제 출제
- **문법 규칙**: 핵심 문법 규칙 설명과 예시
- **오답 분석**: 틀린 문제에 대한 상세한 해설

### 4. 독해 연습
- **지문 독해**: SAT 수준의 독해 지문 제공
- **독해 문제**: 지문 기반 객관식/주관식 문제
- **독해 전략**: 효과적인 독해 방법과 팁 제공

### 5. 학습 관리
- **학습 진도**: 개인별 학습 진행 상황 추적
- **성과 분석**: 강점/약점 분석 및 개선 방향 제시
- **학습 계획**: 개인 맞춤형 학습 스케줄 생성

## 기술적 요구사항

### Frontend
- **React**: 사용자 인터페이스 구현
- **반응형 디자인**: 모바일, 태블릿, 데스크톱 지원
- **PWA**: 오프라인 학습 가능한 Progressive Web App

### Backend (향후 확장)
- **API 설계**: RESTful API로 데이터 관리
- **데이터베이스**: 단어, 문제, 사용자 진도 저장
- **음성 처리**: TTS(Text-to-Speech) 엔진 연동

### 데이터
- **단어 데이터베이스**: SAT 빈출 단어 3000-5000개
- **예문 데이터**: 실제 문장, 뉴스, 문학작품에서 추출
- **문법 문제**: SAT 스타일 문법 문제 은행
- **독해 지문**: 다양한 주제의 SAT 수준 지문

## 사용자 경험 (UX)

### 학습 플로우
1. **레벨 테스트**: 초기 실력 측정
2. **맞춤 학습**: 개인 수준에 맞는 단어/문법 제공
3. **반복 학습**: 틀린 문제 재출제 시스템
4. **진도 확인**: 실시간 학습 진행 상황 확인

### 사용자 인터페이스
- **직관적 디자인**: 학생들이 쉽게 사용할 수 있는 인터페이스
- **시각적 피드백**: 학습 성과의 시각적 표현
- **게이미피케이션**: 포인트, 배지 등을 통한 학습 동기 부여

## 성능 요구사항
- **빠른 로딩**: 페이지 로딩 시간 3초 이내
- **음성 재생**: 1초 이내 발음 재생
- **반응형**: 모든 기기에서 매끄러운 사용자 경험

## 보안 및 개인정보
- **데이터 보호**: 학습자 개인정보 보호
- **안전한 접근**: 적절한 인증 및 권한 관리

## 향후 확장 계획
- **AI 튜터**: 개인 맞춤형 AI 학습 도우미
- **소셜 기능**: 친구와 경쟁, 스터디 그룹
- **모바일 앱**: 네이티브 모바일 애플리케이션 개발
- **다국어 지원**: 한국어 외 다른 언어 학습자 지원
